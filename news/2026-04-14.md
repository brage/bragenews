## Microsoft Fabric

- **[FabCon 2026: Key Announcements](https://www.techment.com/blogs/microsoft-fabric-fabcon-2026-insights/)** — FabCon 2026 marked a strategic shift for Fabric from a data warehousing platform to a full enterprise intelligence layer. Headline features: **Fabric IQ** (semantic layer that understands business terms like "revenue"), **Fabric Graph** (scalable graph database for modelling relationships), **Fabric data agents** (GA — virtual analysts for domain-specific insights), and **Fabric Maps** (geospatial real-time analytics). Also notable: OneLake Catalog Search API (preview) for cross-workspace discovery via code, and a new Git compare-code experience. Read the full March feature summary at [blog.fabric.microsoft.com](https://blog.fabric.microsoft.com/en-us/blog/fabric-march-2026-feature-summary?ft=All) and the [Azure blog summary](https://azure.microsoft.com/en-us/blog/fabcon-and-sqlcon-2026-unifying-databases-and-fabric-on-a-single-data-platform/).

- **[Microsoft acquires Osmos for autonomous data engineering in Fabric](https://blogs.microsoft.com/blog/2026/01/05/microsoft-announces-acquisition-of-osmos-to-accelerate-autonomous-data-engineering-in-fabric/)** — Microsoft acquired Osmos (January 5, 2026), an agentic AI platform that turns raw data into analytics-ready assets in OneLake. Customers reported 50% reduction in dev and maintenance effort. Osmos's standalone products are sunsetting as the tech is absorbed into Fabric — relevant context for any customer relying on Osmos pipelines today.

- **[Databricks at FabCon 2026: Lakebase, Lakeflow Free Tier, Genie Code](https://www.databricks.com/blog/whats-new-azure-databricks-fabcon-2026-lakebase-lakeflow-and-genie)** — Databricks used FabCon to announce **Lakebase GA** (managed serverless Postgres inside the lakehouse, now in 14 Azure regions), **Lakeflow Connect Free Tier** (100 free DBUs/day, ~100M records/workspace/day — zero cost ingestion entry point), and **Genie Code** (agentic dev assistant for notebooks, SQL editors, and Lakeflow pipelines — understands Unity Catalog context and builds pipelines from natural language). Strong signal that Databricks is doubling down on Azure co-sell rather than competing head-on with Fabric.
