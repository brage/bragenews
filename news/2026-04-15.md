# News Digest — 2026-04-15

## Microsoft Fabric

- **[Fabric Eventstream Q1 2026: Enterprise-Ready Real-Time Intelligence](https://blog.fabric.microsoft.com/en-US/blog/whats-new-in-fabric-eventstream-2026-q1-edition)** — Q1 brought three major upgrades to Fabric Eventstream: private network support (VNet/ExpressRoute/on-prem via streaming virtual network data gateway), IoT/database streaming via DeltaFlow Preview (Azure SQL, PostgreSQL) and MQTT v3 connector for edge sensors, and enterprise-grade Kafka security with custom CA and mTLS for Confluent Cloud, Amazon MSK, and Apache Kafka. This makes Eventstream a credible choice for regulated industries like finance and energy that require secure, private data pipelines into OneLake.

- **[Fabric February 2026 Feature Summary](https://blog.fabric.microsoft.com/en-US/blog/fabric-february-2026-feature-summary/)** — February's update delivered improvements across the full Fabric stack: OneLake Catalog governance enhancements, improved developer experiences, and meaningful updates to Data Engineering, Data Factory, and Real-Time Intelligence. Solid incremental release bridging the gap between January's AI-focused drops and the bigger FabCon announcements in March.

- **[Microsoft and Databricks deepen OneLake interoperability](https://blog.fabric.microsoft.com/en-us/blog/microsoft-and-databricks-advancing-openness-and-interoperability-with-onelake?ft=All)** — Microsoft and Databricks announced bidirectional OneLake integration: Unity Catalog tables are mirrored into OneLake as shortcuts (giving Fabric workloads read access to Databricks-managed data), while Databricks can write directly back to OneLake via Delta Lake and ADLS Gen2 APIs. Write timelines were confirmed at FabCon March 2026. Removes the last major friction point for organisations running both platforms side by side — important context when advising customers on hybrid lakehouse architecture.

## AI News

- **[Claude Mythos Preview — Anthropic's specialized cybersecurity model](https://red.anthropic.com/2026/mythos-preview/)** *(April 7–8, 2026)* — Anthropic disclosed Claude Mythos Preview as a gated research model for defensive cybersecurity work (Project Glasswing, invite-only). Benchmarks show a leap over Opus 4.6: 93.9% on SWE-bench Verified (vs. 80.8%), 79.6% on OSWorld, and 97.6% on USAMO 2026. Anthropic states Mythos won't be generally available in its current form, but frames it as the direction for future Mythos-class deployments. Relevant for customers with security operations or vulnerability management use cases.

- **[OpenAI launches GPT-5.4 with native computer use and reasoning levels](https://openai.com/index/introducing-gpt-5-4/)** *(March 5, 2026)* — GPT-5.4 is the first general-purpose OpenAI model with native computer-use capabilities (up to 1M token context) and five discrete reasoning levels (none → xhigh) that developers tune per request. Combines GPT-5.3-Codex coding strength with improved accuracy across enterprise documents. Priced at $2.50/M input tokens. Direct competitor to Claude Opus 4.6 for agentic enterprise workloads; GPT-5.4 Thinking and Pro variants available simultaneously.

- **[Microsoft 365 Copilot March 2026: Cowork, Tasks, Critic](https://techcommunity.microsoft.com/blog/microsoft365copilotblog/what%E2%80%99s-new-in-microsoft-365-copilot--march-2026/4506322)** — Three headline features: **Copilot Cowork** (Copilot executes tasks across M365 apps rather than just suggesting), **Copilot Tasks** (autonomous multi-step task orchestration), and **Critic** (generates a response with one model and reviews it with a second to reduce errors). M365 AI pricing changes effective July 1, 2026. Strong signal that Microsoft is moving Copilot from assistant to autonomous coworker — shapes how Norwegian partners should position M365 Copilot engagements.
